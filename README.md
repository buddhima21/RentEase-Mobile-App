# 📱 RentEase – Rental Management System

RentEase is a full-stack mobile application developed as part of the SE2020 – Web and Mobile Technologies Group Assignment. The system is designed to simplify rental property management by enabling users to manage properties, tenants, bookings, and payments efficiently.

This application follows a MERN-based architecture using React Native for the mobile frontend, Node.js and Express.js for the backend, and MongoDB as the database.

---

## 🚀 Tech Stack

- **Frontend:** React Native (Expo)
- **Backend:** Node.js + Express.js
- **Database:** MongoDB (MongoDB Atlas)
- **Authentication:** JWT (JSON Web Tokens)
- **API:** RESTful API

---

## 📌 Project Overview

The RentEase system allows:

- Users to register and log in securely
- Property owners/admins to manage rental properties
- Users to browse and view available properties
- Booking and rental management
- Secure API communication between mobile app and backend

---

## 🔐 Core Features

### 1. Authentication System
- User Registration
- Login System
- Password Hashing
- JWT-based Authentication
- Protected Routes

### 2. Property Management
- Add new properties
- View property listings
- Update property details

### 3. Booking Management
- Create booking requests
- View booking history
- Approve/Reject bookings
- Cancel bookings

### 4. Mobile Application Features
- Clean and responsive UI
- Navigation system
- Form validation
- API integration
